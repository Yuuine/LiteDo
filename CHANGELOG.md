# LiteDo 更新日志

本文档记录项目的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.1.2] - 2026-03-14

### Fixed

- 修复了日历组件中未来日期可被选择的问题。此前用户切换到未来月份时，可以点击选择未来日期，与产品需求"仅允许查看和选择当前及过去的日期"不符。现在所有未来日期均显示为灰色禁用状态
- 修复了消息提示（Toast）显示位置在右侧的问题，现已改为顶部居中显示，视觉焦点更集中

### Added

- 新增公共组件库，包括遮罩层组件、按钮组件和优先级标签组件，提升界面一致性
- 新增 Composables 组合式函数，包括设置管理和主题管理，优化代码复用性

### Changed

- 优化了任务排序逻辑，代码更简洁高效

### Removed

- 移除了未使用的 Modal 组件
- 移除了未使用的 API 函数和类型定义

## [1.1.1] - 2026-03-14

### Added

- 新增任务优先级系统，支持高/中/低三级优先级设置
- 新增优先级全局开关，可在设置面板中启用或禁用优先级功能
- 新增按优先级排序功能，高优先级任务自动排在前面
- AI 解析功能现支持自动识别任务优先级（可识别"高优先"、"紧急"等关键词）
- 新增任务内容编辑功能，双击任务或点击编辑按钮进入编辑模式
- 编辑任务时失焦自动保存，按 ESC 键取消编辑
- AI 解析对话框新增首次使用引导提示

### Changed

- 重新设计了任务输入栏，将输入框与优先级选择器合并为一体
- 优化了优先级选择器交互，改为下拉菜单形式
- 简化了顶部状态栏布局
- 强化了筛选标签页，新增数字角标显示各状态任务数量
- 压缩了列表项高度约 30%，提升信息密度
- 优化了列表项布局，主要内容靠左对齐
- 移除了卡片背景，改用分割线，视觉更简洁
- 优先级标签现使用颜色区分（高-红色、中-橙色、低-绿色）

### Fixed

- 修复了筛选标签数字角标在高分屏下的锯齿显示问题

## [1.1.0] - 2026-03-13

### Added

- 新增 AI 智能解析功能，支持使用 OpenAI API（兼容 DeepSeek、Claude 等）解析自然语言生成待办事项
- 新增自定义 AI 模型配置，支持添加、编辑、删除多个模型配置
- API Key 采用加密存储，保障用户数据安全
- 新增删除任务成功提示
- 更新了桌面应用图标和项目 Logo

### Changed

- 优化了任务排序逻辑，已完成任务自动排在后面
- 优化了日历组件交互，月份切换更流畅
- 添加了 GitHub Actions 自动化构建和发布流程

### Fixed

- 修复了若干已知问题和 UI 显示问题

## [1.0.0] - 2026-03-13

### Added

- 新增任务管理核心功能：添加、编辑、删除任务
- 新增任务完成状态切换，点击任务卡片即可切换
- 新增日期筛选功能，支持按日期查看任务
- 新增任务数量实时统计（全部/待办/完成）
- 新增日历选择器，支持月份导航和日期选择
- 新增主题定制功能，提供 9 种预设主题颜色（靛蓝、紫罗兰、玫红、红色、橙色、黄色、绿色、青色、蓝色）
- 新增自定义颜色选择器，支持任意颜色
- 新增待办事项字数限制设置（10-200 字）
- 新增开机自启动设置选项（预留）
- 新增日志系统，包括操作日志、调试日志和系统日志

### Changed

- 优化了待办事项卡片交互，整个卡片可点击切换完成状态
- 优化了已完成任务的视觉效果（更淡的颜色、更粗的删除线）
- 悬停时显示删除按钮，提升界面简洁度
- 优化了日历组件，新增当前日期高亮和选中日期标记
- 设置面板采用模态窗口设计，分组展示设置项

### Fixed

- 修复了调试日志查看器无法打开的问题
- 修复了日历组件日期计算问题
- 修复了设置保存时的 undefined 显示问题
- 修复了事件监听器内存泄漏问题
